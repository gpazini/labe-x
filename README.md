## 💻 About
Projeto de uma plataforma de viagens espacial, com a possibilidade do usuário olhar o catalogo, escolher uma opção e se candidatar para a viagem.
Na área administrativa (privada), o usuário pode: criar novas viagens, deletar as existentes, aprovar ou reprovar candidatos para as viagens.

## 📥 Install 
Faça um clone desse repositório e acesse o diretório.

```
$ git clone https://github.com/gpazini/labe-x.git
```

```
# Instalando as dependências
$ npm install styled-components
$ npm install axios
$ npm install react-router-dom@5.3
```

```
# Executanto aplicação
$ npm run start

```
